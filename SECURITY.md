# Segurança

Não abra Issue pública para vulnerabilidades.

## Como reportar
Envie um relato privado com:
- descrição do problema
- passos para reproduzir
- impacto estimado
- sugestão de correção (se houver)

## SLA (informal)
Vou responder assim que possível e priorizar correções que afetem:
- execução remota, vazamento de dados, negação de serviço.
