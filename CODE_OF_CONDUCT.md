# Código de Conduta

Este projeto adota um padrão simples: respeito e profissionalismo.

## Esperado
- Tratar todos com respeito.
- Discordar de ideias sem atacar pessoas.
- Feedback técnico, objetivo e verificável.

## Não tolerado
- Assédio, humilhação, discriminação, ameaças.
- Conteúdo ofensivo, sexual ou agressivo.
- Doxxing (expor dados pessoais).

## Aplicação
O mantenedor pode: advertir, remover conteúdo, bloquear usuário e rejeitar PRs, conforme gravidade.
